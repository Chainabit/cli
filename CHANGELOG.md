## [0.1.0]

### Added

- Initial public release of the Chainabit CLI.
- `auth` — browser device-approval login, logout, whoami, token refresh, and developer API key management.
- `workspace` — pin and manage workspace context.
- `connectors` — install, authenticate, test, and execute connectors.
- `ai` — run AI sessions, suggestions, and Chao conversations.
- `--json` output mode for scripting and CI/CD.
